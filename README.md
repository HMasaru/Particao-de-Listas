# Partição de Listas

> 🎓 **Projeto Acadêmico**
> Repositório destinado aos projetos e atividades desenvolvidos para o curso de **Ciência da Computação da PUC**.

Implementação em **C** de algoritmos para particionamento de listas. 

Este projeto explora conceitos fundamentais de **Estruturas de Dados** e **Algoritmos**, focando na manipulação de listas (arrays ou listas encadeadas). O particionamento é uma técnica essencial presente em algoritmos de ordenação de alta performance (como o QuickSort) e em operações de divisão de dados baseadas em critérios específicos (como um elemento pivô).

## 📂 Estrutura do Projeto

Os códigos-fonte estão contidos dentro do diretório `Partição de Listas/`. O desenvolvimento foi feito com foco direto na manipulação de ponteiros, alocação de memória e lógica algorítmica pura oferecida pela linguagem C.

## 🚀 Como Compilar e Executar

Para rodar este projeto, é necessário ter um compilador C (como o `gcc`) instalado no seu ambiente.

Abra o terminal e siga os passos abaixo:

```bash
# 1. Entre no diretório do projeto
cd "Partição de Listas"

# 2. Compile o código-fonte
gcc nome_do_arquivo.c -o particao_listas

# 3. Execute o programa (Linux ou macOS)
./particao_listas

# 3. Execute o programa (Windows)
particao_listas.exe
